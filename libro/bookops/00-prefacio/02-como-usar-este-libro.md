# Como usar este libro

Este libro esta armado para estudiar un parcial a desarrollar, no para reemplazar la lectura de Freud.

La materia tiene tres espacios: teorico, practico y seminario. El problema es que esos espacios no separan textos de manera limpia: un mismo texto puede reaparecer con otro uso. Por eso el libro tiene dos recorridos:

- Un recorrido por cursada: que se vio en cada espacio.
- Un recorrido conceptual integrado: como se enlazan los conceptos.

## Regla de escritura para el parcial

Cada respuesta deberia tener:

1. Una tesis inicial.
2. Conceptos precisos.
3. Texto o zona de Freud.
4. Referente clinico o ejemplo si corresponde.
5. Cierre que retome la pregunta.

No alcanza con nombrar conceptos: hay que articularlos.
