## Ruta recomendada

Para estudiar rapido:

1. Leer `05-entrenamiento-parcial/01-prioridades.md`.
2. Practicar `05-entrenamiento-parcial/02-respuestas-modelo.md`.
3. Resolver un simulacro de `05-entrenamiento-parcial/04-simulacros.md`.
4. Volver a la sintesis integrada para los temas flojos.

Para entender el recorrido:

1. Leer el mapa de cursada.
2. Leer la sintesis integrada.
3. Bajar a los casos cuando haga falta desarrollar escenas o recorridos clinicos.
4. Usar el mapa de textos para ubicar rapido escritos, problemas y ejemplos.
5. Usar entrenamiento parcial como evaluacion.
