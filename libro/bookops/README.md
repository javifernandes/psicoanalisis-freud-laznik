## Ruta recomendada

Para estudiar rapido:

1. Leer `03-entrenamiento-parcial/01-prioridades.md`.
2. Practicar `03-entrenamiento-parcial/02-respuestas-modelo.md`.
3. Resolver un simulacro de `03-entrenamiento-parcial/04-simulacros.md`.
4. Volver a la sintesis integrada para los temas flojos.

Para entender el recorrido:

1. Leer el mapa de cursada.
2. Leer la sintesis integrada.
3. Usar entrenamiento parcial como evaluacion.
